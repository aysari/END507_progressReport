Kodu Çalıştırma için Yapılması Gerekenler
	1) END507_project dosyası .zip uzantılı indirilir.
	2) END507_project-main.zip dosyasının içindeki END507_project dosyası ECLIPSE'in directory'sine kaydedilir.
	3) ECLIPSE programı çalıştırılır.
	4) File -> Open Project From File System tıklanır.
	5) Package Explorer sekmesinden END507_project -> src -> END507_projectjava açılır.
	6) Kod runlanır.
